Technology used

Spring Boot, Spring Cloud, Spring Data JPA, Spring Security

Maven, Hibernate, Lombok, Jackson, Eureka-server, Zuul-gateway

Angular, Bootstrap

MySql/Postgres, Liquibase

Main features

Two microservices (user and topic) are connected with each other
Ability to change user info + update user password
Ability to add post and post details for registered user
User who aren't registered can view posts+details but can't create new
User also can update/delete own posts
User can delete own account
User can insert profile picture (It will be saved to Firebase storage)
User must submit account after registration by email
Ability to reset password by user request (the new will be sent to user email)
Angular guards + roles access to some links
Ability for admin enable/disable users + search user in "manage page"
Pagination for list of users with limit amount per page
Using a lot of Angular features as: dialogs, alerts, slice version of array
